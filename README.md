# robosys2023
ロボットシステム学の練習リポジトリ

# plusコマンド
[![test](https://github.com/shuheeey/robosys2023/actions/workflows/test.yml/badge.svg)](https://github.com/shuheeey/robosys2023/actions/workflows/test.yml)

## インストール方法
クローンする
```bash
git clone https://github.com/shuheeey/robosys2023.git
```
robosys2023に移動する
```bash
cd robosys2023
```

## 使い方

標準入力から読み込んだ数字の計算をする。
### 実行例
```bash
seq 5 | ./plus
```
### 実行結果
・和・積・平均値・逆数和・２のべき乗
```bash
合計値:15
乗算値:120
平均値:3.0
逆数和:137/60
2のべき乗:32
```
